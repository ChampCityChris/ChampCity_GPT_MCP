Cloudflare / public endpoint

Read-only first, then carefully enable writes.
  1. Configure an HTTP auth token
  2. Keep unauthenticated local mode disabled
  3. Keep write mode Off for first testing
  4. Start the local HTTP MCP server on 127.0.0.1:3333
  5. Configure Cloudflare Tunnel for your public MCP hostname
  6. Verify the public endpoint before ChatGPT registration
  7. Register the public MCP endpoint in ChatGPT Connectors/Create
